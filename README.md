WISH ONLINE STORE
 Project Overview

Wish Online Store is a simple responsive e-commerce website built using HTML, CSS, and Bootstrap 5.
It showcases a modern landing page with a navigation bar, hero section, product grid, and footer.

FEATURES;
Responsive navigation bar (Bootstrap 5)
Hero section with call-to-action button
Product grid layout using Bootstrap cards
Hover effects on product cards
Clean footer with contact information
Mobile-friendly design

 TECHNOLOGIES USED
HTML5
CSS3
Bootstrap 5
